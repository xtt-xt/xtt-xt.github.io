项目目的

这个测试网站旨在帮助你快速验证 GitHub Pages 的部署是否成功，并提供一个可立即使用的模板。

🎯 核心特性

  • 响应式设计：适应手机、平板和电脑屏幕
  • 简洁导航：页面间可轻松跳转
  • 现代界面：使用CSS Flexbox布局
  • 即时反馈：显示部署时间和状态

🛠️ 技术细节

HTML5

语义化标签，现代化结构

CSS3

Flexbox布局，响应式设计

JavaScript

基础交互功能

🚀 如何使用

  1. 将这三个文件上传到你的GitHub Pages仓库
  2. 访问 https://你的用户名.github.io
  3. 如果看到此页面，说明部署成功！
  4. 开始自定义内容，创建属于你自己的网站

💡 自定义提示

要修改此网站：

  1. 用文本编辑器打开 .html 文件更改内容
  2. 编辑 style.css 调整颜色、字体等样式
  3. 使用返回首页测试导航是否正常
